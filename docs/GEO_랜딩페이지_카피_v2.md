## [DEFINITION] GEO가 뭔가

**섹션 헤드라인**
GEO — Generative Engine Optimization

**서브헤드**
SEO가 검색엔진 순위를 관리하듯,
GEO는 AI 답변 안에서의 브랜드 위치를 관리합니다.

**3개 핵심 개념 카드**

### 엔티티 (Entity)
AI 지식그래프 안에 존재하는 '것'.
AI가 브랜드에 대해 알고 있는 것 전체.

### 인텐트 (Intent)
사용자가 AI에게 원하는 행위.
"추천해줘", "비교해줘", "가격 알려줘" — 우리가 등장해야 할 순간.

### GEO
인텐트에 맞게 엔티티를 채우는 작업.
AI의 지식 공백을 우리 정보로 채워, 답변 안에 우리가 포함되도록 설계합니다.

## [HOW] 어떻게 작동하는가

**섹션 헤드라인**
GEO는 5단계로 작동합니다

**5단계 플로우**

1. **진단**
   현재 AI 가시성 측정. 플랫폼별 인용률, SOV, 경쟁사 점유율.

2. **인텐트 설계**
   우리 브랜드가 노출되어야 할 쿼리를 정의합니다.
   추천 / 비교 / 가격 / 정보탐색 / 전환.

3. **엔티티 구축**
   인텐트별 GEO 콘텐츠 생성.
   통계·출처·수치 — AI가 신뢰하는 방식으로 작성합니다.

4. **기술 최적화**
   Schema.org JSON-LD, Canonical, Sitemap.
   콘텐츠가 아무리 좋아도 기술이 빠지면 AI가 읽지 못합니다.

5. **모니터링**
   매일 자동 크롤링 → 인용 현황 누적 → 엔티티 그래프 갱신.
   수치로 확인하고, 수치로 개선합니다.
